# 🚀 Axora - Multipurpose Discord Bot

Official management infrastructure for **Pixora Agency**. Developed by **Xicx_**.

## ✨ Features
* **🛡️ Antinuke & Security:** Advanced protection against server nukers.
* **⚡ No-Prefix (NP):** Whitelisted users can use commands without `$`.
* **🔨 Moderation:** Full suite of 100+ admin tools.
* **📊 Management:** Custom embeds and crew setup for creators.

## 🚀 How to Run (GitHub Actions)
1. Go to **Settings > Secrets > Actions**.
2. Add a secret named `TOKEN` with your Bot Token.
3. Go to the **Actions** tab.
4. Select **Axora Bot Runner** and click **Run workflow**.
